欢迎使用 CycleMind —— 一款 AI 驱动的软件设计平台。本文将带你快速了解核心工作流，从需求输入到完整设计文档输出，只需五分钟。

什么是 CycleMind？

CycleMind 是一个将自然语言需求转化为专业软件设计文档的平台。它内置五大专业 Agent，各司其职：

  • Requirement Agent —— 解析和结构化你的需求描述
  • Design Agent —— 生成系统架构图（Mermaid 格式）
  • ER Agent —— 自动设计数据库 ER 图
  • API Agent —— 提取并生成 RESTful API 规范
  • Plan Agent —— 输出 Sprint 级开发计划

快速开始

按照以下步骤，即可体验完整的 AI 设计流程：

第一步：注册账号

访问首页，点击「免费注册」按钮，使用邮箱或 GitHub 账号完成注册。

第二步：创建会话

登录后进入 Dashboard，点击「新建会话」。在输入框中用自然语言描述你的项目需求，例如：

我需要一个在线书店系统，支持用户注册登录、图书浏览与搜索、购物车、订单管理和支付集成。

第三步：查看生成结果

提交需求后，Orchestrator 会自动调度五大 Agent 并行工作。你可以在工作区实时查看每个 Agent 的生成进度，包括：

  • 结构化需求文档
  • 系统架构图（可交互的 Mermaid 图表）
  • 数据库 ER 关系图
  • 完整的 API 接口规范
  • Sprint 开发计划与时间线

追加需求与版本追踪

CycleMind 支持增量迭代。在同一个会话中追加新的需求描述，系统会自动对比差异、生成变更日志，并保留完整的版本历史。每个版本都可以单独查看和对比。
